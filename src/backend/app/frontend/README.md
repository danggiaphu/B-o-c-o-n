# Streamlit Frontend

Thu muc nay chua giao dien web Streamlit (frontend) cho he thong du doan Thuoc-Benh.

## Chay backend API

```bash
pip install -r requirements.txt
python backend/main.py
```

## Chay frontend

```bash
streamlit run frontend/streamlit_app.py
```

Hoac chay truc tiep file frontend trong package:

```bash
streamlit run backend/app/frontend/streamlit_app.py
```
